Hello! Guys!

我们在运用电脑工作的时候，总是避免不了要用鼠标点击，而不能用意念操作，大部分小伙伴可能也感受到了，用鼠标点实在是太慢了，有什么提高效率的好技巧呢？

如果我们使用快捷键，效率会更高！

什么是快捷键？

快捷键，字面意思上看，就是操作某功能快速的按键。下面是常用的快捷键，熟练后效率会有大大飞跃。

部分小伙伴可能都会感受到，大部分软件的快捷键几乎完全一致，这似乎是个趋势。

基本快捷键

  • Ctrl + Z撤销操作(Undo)
  • Ctrl + Y恢复操作(Redo)
  • Ctrl + S保存(Save)
  • Ctrl + F查找(Find)
  • Ctrl + R替换(Replace)
  • Ctrl + C复制(Copy)
  • Ctrl + V粘贴(Paste)
  • Ctrl + X剪切(Cut)
  • Ctrl + A全选(Select All)
  • Shift切换输入法
  • Ctrl + Shift切换中英文输入法
  • Tab在文本编辑器下将插入一段可能是空格的空白，在应用程序将改变获得焦点的对象

光标快捷键

  • Insert点击后进入改写模式，再点一次退出。改写模式下后方的文字将被输入的内容覆盖
  • Delete删掉光标后的字符
  • Shift + ← / → (方向键)光标向左/向右选中一个字符
  • Ctrl + Shift + ← / → (方向键) 光标向左/向右选中一个单词
  • Shift + Home光标从当前位置一直选中到行首
  • Shift + End光标从当前位置一直选中到行尾
  • (鼠标双击文本)快速选中鼠标位置单词
  • Home光标跳转至行首
  • End光标跳转至行尾
  • Ctrl + ← / → (方向键) 光标向左/向右跳转一个单词
  • Ctrl + Home光标跳转至首行首个字符前面(文件开头)
  • Ctrl + End光标跳转至末行最后一个字符后面(文件结尾)

窗口快捷键

  • F5（浏览器）刷新界面
  • F6（浏览器）使地址栏接受焦点
  • F11全屏当前对话窗口
  • F6（浏览器）使地址栏接受焦点

代码编辑器常用快捷键

  • Ctrl + K + C(Visual Studio专用)注释文本
  • Ctrl + K + U(Visual Studio专用)取消注释文本
  • Ctrl + /注释或取消掉注释
  • F5运行代码
